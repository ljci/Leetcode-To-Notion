üìö Quick Setup Guide

Step 1: Create Notion Integration

  1. Go to notion.so/my-integrations
  2. Click "+ New integration"
  3. Name it "LeetCode Tracker"
  4. Copy the Integration Token (starts with secret_)

Step 2: Create Database

  1. Create a new page in Notion
  2. Add a Database (Full page)
  3. Name it "LeetCode Problems"
  4. Add these properties:
    • Problem Name (Title)
    • Number (Number)
    • Difficulty (Select: Easy, Medium, Hard)
    • Status (Select: Solved, Review, Mastered)
    • URL (URL)
    • Tags (Multi-select)
    • Time Spent (Text)
    • Date Added (Date)
    • Language (Select: Python, JavaScript, Java, C++, SQL)
    • Needed Hint (Checkbox)
    • Can Redo (Checkbox)
    • Notes (Text)

Step 3: Connect Database

  1. In your Notion database, click "..." (top right)
  2. Select "Add connections"
  3. Choose your "LeetCode Tracker" integration
  4. Copy the Database ID from the URL
  5. Paste both Token and Database ID above
  6. Click "Test Connection" to verify
  7. Click "Save Settings" when done
